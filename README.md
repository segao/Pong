# Pong
Recreation of the classic arcade game Pong using Python. One or two players compete until a player reached 11 points. Pygame is required to run.

# Usage
```
python3 pong.py
```

# Controls
Player One:
```
      ___
     | W |
 ___  ___ ____
| A || S |  D |
---------------
```

Player Two:

```
      ___
     | ^ |
 ___  ___ ____
| < || v |  > |
---------------
```
